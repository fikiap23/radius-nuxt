# Radius — Project Management Platform

## Core Concept

Radius adalah platform manajemen kerja modern untuk:

- task management
- sprint planning
- team collaboration
- issue tracking
- productivity monitoring

Fokus:

- clean
- cepat
- developer friendly
- modern UI
- scalable untuk startup sampai enterprise
